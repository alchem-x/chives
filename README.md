# 韭菜观察 Chives Watch

- Node.js 21+

### Setup

Add `.env` file

```ini
# .env
SNOWBALL_TOKEN=
CW_TOKEN=
BARK_API=
```

### Install deps & build

```sh
pnpm install
pnpm build
```

### Dev

```sh
pnpm start
```

### Run

```sh
pnpm serve
```

### Deploy with Docker

```sh
docker compose up -d
```
